# Earth Texture Sources

Low-resolution Earth maps were downloaded from Planet Pixel Emporium:

https://planetpixelemporium.com/earth8081.html

Files used by the renderer:

- `earthmap1k.jpg` - color map
- `earthbump1k.jpg` - elevation bump map
- `earthspec1k.jpg` - ocean specular map
- `earthlights1k.jpg` - night lights emissive map
- `earthcloudmap.jpg` - cloud color map
- `earthcloudmaptrans.jpg` - cloud transparency map
